## Product storage

### Installation:

#### 1. Создать файл .env из .env.example
#### 2. Заранить команды
    - composer update
    - npm install
#### 3. Сгенерировать ключ
    - php artisan key:generate
#### 4. Команда для создания бд и мираций
    - php artisan migrate
#### 5. Команда для создания тестовых данных
    - php artisan app:refresh

#### 6. Запуск
    - php artisan serve
    - npm run dev
